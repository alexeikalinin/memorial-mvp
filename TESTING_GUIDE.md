# Руководство по тестированию vspomin.ai

## ✅ Проект запущен!

Оба сервиса работают:
- **Backend**: http://localhost:8000 ✅
- **Frontend**: http://localhost:5173 ✅

## 🧪 Пошаговое тестирование

### Шаг 1: Откройте Frontend

Откройте в браузере: **http://localhost:5173**

Вы должны увидеть главную страницу с кнопкой "Создать мемориал".

### Шаг 2: Создайте мемориал

1. Нажмите "Создать мемориал"
2. Заполните форму:
   - **Имя**: "Тестовый мемориал" (обязательно)
   - **Описание**: "Это тестовый мемориал для проверки функционала"
   - **Дата рождения**: 01.01.1950 (опционально)
   - **Дата смерти**: 01.01.2020 (опционально)
   - **Публичный мемориал**: оставьте выключенным
3. Нажмите "Создать мемориал"

**Ожидаемый результат**: Вы будете перенаправлены на страницу мемориала с тремя вкладками.

### Шаг 3: Загрузите медиа

1. Перейдите на вкладку **"Медиа"**
2. Нажмите **"Загрузить файл"**
3. Выберите изображение (JPG, PNG) или видео (MP4)
4. Дождитесь загрузки

**Ожидаемый результат**: 
- Файл появится в галерее
- Для изображений автоматически создадутся миниатюры
- Для видео создастся превью

**Проверьте**:
- Изображения отображаются корректно
- Можно просмотреть оригинал
- Миниатюры работают

### Шаг 4: Добавьте воспоминания

1. Перейдите на вкладку **"Воспоминания"**
2. Нажмите **"Добавить воспоминание"**
3. Заполните форму:
   - **Заголовок**: "Детство" (опционально)
   - **Текст**: "Этот человек родился в небольшом городе. В детстве любил читать книги и играть на улице с друзьями. Был очень любознательным и всегда задавал много вопросов."
4. Нажмите **"Сохранить"**

**Ожидаемый результат**: 
- Воспоминание появится в списке
- Будет видна дата добавления
- Автоматически создастся embedding (в фоне, через worker)

**Добавьте еще 2-3 воспоминания** с разными темами:
- О работе
- О семье
- О хобби

### Шаг 5: Проверьте API документацию

Откройте: **http://localhost:8000/docs**

**Проверьте**:
1. Все endpoints отображаются
2. Можно попробовать выполнить запрос прямо в браузере
3. Схемы данных корректны

**Попробуйте**:
- `GET /api/v1/health/` - должен вернуть `{"status":"ok"}`
- `GET /api/v1/memorials/{id}` - получить созданный мемориал

### Шаг 6: Чат с аватаром (требует API ключи)

⚠️ **Требует настройки**: `OPENAI_API_KEY` и `PINECONE_API_KEY` в `.env`

1. Перейдите на вкладку **"Чат с аватаром"**
2. Задайте вопрос: "Расскажи о детстве этого человека"
3. Нажмите "Отправить"

**Ожидаемый результат** (если настроены ключи):
- ИИ-аватар ответит на основе добавленных воспоминаний
- Будут показаны источники (какие воспоминания использованы)
- Опционально: можно включить генерацию аудио

**Без API ключей**:
- Покажет сообщение об ошибке или что нет информации

### Шаг 7: Оживление фото (требует API ключи)

⚠️ **Требует настройки**: `DID_API_KEY` или `HEYGEN_API_KEY` в `.env`

1. Перейдите на вкладку **"Медиа"**
2. Найдите загруженное фото
3. Нажмите кнопку **"Оживить фото"**

**Ожидаемый результат** (если настроены ключи):
- Задача анимации будет запущена
- Через некоторое время фото будет заменено на видео
- Появится бейдж "Оживлено"

**Без API ключей**:
- Покажет ошибку при попытке запуска

## 🔍 Проверка через API

### Создание мемориала через curl

```bash
curl -X POST http://localhost:8000/api/v1/memorials/ \
  -H "Content-Type: application/json" \
  -d '{
    "name": "API Test Memorial",
    "description": "Created via API",
    "is_public": false
  }'
```

### Загрузка медиа через curl

```bash
curl -X POST http://localhost:8000/api/v1/memorials/1/media/upload \
  -F "file=@/path/to/photo.jpg"
```

### Добавление воспоминания через curl

```bash
curl -X POST http://localhost:8000/api/v1/memorials/1/memories \
  -H "Content-Type: application/json" \
  -d '{
    "title": "Воспоминание",
    "content": "Текст воспоминания о человеке..."
  }'
```
