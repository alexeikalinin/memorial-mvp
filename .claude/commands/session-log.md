Работа с логом сессий проекта memorial-mvp.

Лог находится по пути:
`/Users/alexei.kalinin/.claude/projects/-Users-alexei-kalinin-Documents-VibeCoding-memorial-mvp/memory/session_log.md`

Определи, что нужно сделать, исходя из запроса пользователя:

---

**Если пользователь говорит "покажи лог", "что мы делали", "с чего начать", "где мы остановились":**
1. Прочитай session_log.md полностью
2. Выведи последние 3 записи в читаемом виде
3. Укажи задачи со статусом "в процессе" или "осталось сделать"
4. Предложи продолжить с места остановки

---

**Если пользователь говорит "найди решение для [проблемы]" или "мы уже это решали":**
1. Прочитай session_log.md
2. Найди записи где упоминается похожая проблема в разделе "Проблемы и решения"
3. Выведи найденное решение с датой и контекстом
4. Если не нашёл — так и скажи

---

**Если пользователь говорит "запиши", "сохрани прогресс", "обнови лог":**
1. Прочитай session_log.md
2. Составь новую запись в формате:
```
## [YYYY-MM-DD] Краткое название задачи
**Статус:** завершено | в процессе | заблокировано
**Что делали:** ...
**Проблемы и решения:** ...
**Изменённые файлы:** ...
**Осталось сделать:** ...
```
3. Добавь запись в НАЧАЛО файла (после заголовка и строки-разделителя)
4. Не удаляй старые записи

---

**Если пользователь говорит "покажи все открытые задачи" или "что осталось сделать":**
1. Прочитай session_log.md
2. Собери все пункты из разделов "Осталось сделать" по всем записям
3. Отфильтруй задачи, которые уже отмечены как завершённые в более поздних записях
4. Выведи актуальный список
