# Memorial MVP — План тестирования

> Последнее обновление: 2026-03-28
> Статус: живой документ — обновляется при добавлении новых фич

## Как использовать этот документ

- **Ручное тестирование:** пройди по блокам 1-13 перед релизом
- **Авто-тесты backend:** `cd backend && pytest` — покрывают блоки 1-7
- **E2E тесты:** `cd e2e && npx playwright test` — покрывают блоки 1-3, 6, 8
- **Скилл авто-запуска:** `/run-tests` — запускает всё и выдаёт отчёт
- **Скилл авто-фикса:** `/test-and-fix` — тесты → фикс багов → повтор → отчёт

## Регрессионный smoke-тест (после каждого деплоя)

Минимальный набор для проверки работоспособности:

```bash
# 1. Health check
curl http://localhost:8000/health

# 2. Зарегистрироваться → войти → получить токен
# 3. Создать мемориал → добавить воспоминание → спросить аватара → ответ с источником
# 4. Открыть публичный мемориал без авторизации
```

Автоматически выполняется через `/run-tests`.
