Схема: поколения + две линии + перекрёстный брак

Условная карикатура по рекомендации: горизонтальные этажи = поколения (сверху старше), цвет карточки = линия семьи А или В, золотая рамка = супруги «мост» между линиями. Откройте файл в браузере (двойной клик или перетащите вкладку).
