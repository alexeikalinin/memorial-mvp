# Монетизация Memorial MVP — POC (выбор модели и лимиты)

Документ фиксирует **рекомендуемую гибридную модель** и **черновые лимиты по AI-фичам** для последующей реализации биллинга (после стабилизации UI доступа и ролей, см. `HANDOFF.md`).

## Выбранная модель

**Гибрид: Freemium + подписка + разовая опция «за мемориал».**

| Компонент | Назначение |
|-----------|------------|
| **Free** | Воронка: один активный мемориал, базовый чат, жёсткие лимиты на дорогие фичи (голос, анимация). |
| **Подписка (Plus / Family)** | Предсказуемый MRR: несколько мемориалов, семейный RAG, повышенные квоты, хранилище. |
| **Разовая оплата (Lifetime memorial)** | Альтернатива подписке для одной страницы «навсегда» (сценарий QR на памятнике, пользователи без желания платить ежемесячно). |

**Обоснование:** себестоимость доминируют LLM, TTS и видео-рендеры; бесплатный тариф обязан ограничивать именно их. Разовая опция снижает трение в эмоционально тяжёлой нише.

## Лимиты по AI-фичам (черновик для инженерной реализации)

Значения — стартовая точка для калибровки по метрикам и марже; подлежат A/B и пересмотру.

### Тариф Free

| Фича | Лимит | Примечание |
|------|--------|------------|
| Чат с аватаром (RAG) | 30 сообщений / мемориал / месяц | Только текст; без `include_audio`. |
| Семейный RAG (`include_family_memories`) | Выключено | Включается в платных тарифах. |
| Озвучка (ElevenLabs) | 0 | Кнопка «голос» скрыта или с апселлом. |
| Оживление фото (D-ID / HeyGen) | 0 рендеров / месяц | Или 1 пробный при верификации email — на усмотрение продукта. |
| Эмбеддинги новых воспоминаний | До 50 новых текстовых воспоминаний / мемориал (накопительно в free) | Защита от спама; точное число — по нагрузке на Qdrant/OpenAI. |

### Тариф Plus (подписка)

| Фича | Лимит |
|------|--------|
| Чат | 500 сообщений / мемориал / месяц (агрегировать по аккаунту — вариант). |
| Семейный RAG | Включён до N связанных мемориалов (например 10). |
| Озвучка | ~15 000 символов TTS / месяц на аккаунт (связать с лимитами ElevenLabs). |
| Оживление фото | 10 рендеров / месяц на аккаунт. |
| Медиа-хранилище | Например 5 GB на аккаунт или по мемориалу — отдельное решение. |

### Тариф Family (расширение Plus)

- Увеличение лимитов чата/TTS/рендеров (например ×2 к Plus).
- Больше мемориалов под одной подпиской (например до 10).

### Разовая покупка «Lifetime» (один мемориал)

- Эквивалент **Plus по лимитам только для выбранного `memorial_id`**, без срока действия подписки (или с длительным горизонтом, например 50 лет — юридически уточнить).
- Перенос «на другой мемориал» не входит или платная услуга.

## Метрики и защита

- **Metering:** счётчики на уровне `user_id` + `memorial_id` для чата, TTS, анимации; отдельно — вызовы embedding при создании/редактировании памяти.
- **Злоупотребления:** rate limit по IP на публичном чате; капча при массовой регистрации при необходимости.

4. Связать внутренние лимиты с реальной себестоимостью (см. `.claude/commands/check-quotas.md`).
