# vspomin.ai - Frontend

React frontend для веб-сервиса хранения цифровой памяти и создания ИИ-аватаров.

## 🚀 Быстрый старт

### Предварительные требования

- Node.js 18+ и npm/yarn
- Backend должен быть запущен на http://localhost:8000

### Установка

1. Перейдите в директорию frontend:
```bash
cd frontend
```

2. Установите зависимости:
```bash
npm install
# или
yarn install
```

3. Запустите dev сервер:
```bash
npm run dev
# или
yarn dev
```

Приложение будет доступно на http://localhost:5173

## 📁 Структура проекта

```
frontend/
├── src/
```

## 🔧 Конфигурация

### Переменные окружения

Создайте файл `.env` в директории `frontend/`:

```env
VITE_API_URL=http://localhost:8000/api/v1
```

По умолчанию используется прокси через Vite (см. `vite.config.js`).

## 📝 Основные функции

### Создание мемориала

- Форма с полями: имя, описание, даты рождения/смерти
- Публичный/приватный мемориал

### Загрузка медиа

- Загрузка изображений, видео, аудио
- Автоматическое отображение в галерее
- Кнопка "Оживить фото" для изображений

### Воспоминания

- Добавление текстовых воспоминаний
- Отображение списка воспоминаний
- Автоматическое создание embeddings

### Чат с аватаром

- Задавание вопросов об человеке
- Ответы на основе RAG
- Опциональная генерация аудио
- Отображение источников

## 🛠 Технологии

- **React 18** - UI библиотека
- **React Router** - маршрутизация
- **Vite** - сборщик и dev сервер
- **Axios** - HTTP клиент
- **CSS** - стилизация

## 📚 API Endpoints

Frontend использует следующие endpoints:

- `GET /api/v1/memorials/{id}` - получить мемориал
- `POST /api/v1/memorials/` - создать мемориал
- `POST /api/v1/memorials/{id}/media/upload` - загрузить медиа
- `GET /api/v1/memorials/{id}/media` - получить медиа
- `POST /api/v1/memorials/{id}/memories` - добавить воспоминание
- `POST /api/v1/ai/avatar/chat` - чат с аватаром
- `POST /api/v1/ai/photo/animate` - оживить фото
- `GET /api/v1/media/{id}` - получить медиа-файл
